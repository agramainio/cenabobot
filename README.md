# cenabobot

Telegram meal suggestion bot.

Core product rule:
- Suggest one trusted meal at a time.
- Recipes come only from a predefined catalogue.
- AI may parse requests but must not invent recipes.
- Users can tap Next idea, Show recipe, Shopping list, Save, or Not this.

Version 1:
- Telegram private group support
- Button-first interface
- Postgres database
- Predefined recipe catalogue
- One suggestion at a time
- Shopping list
